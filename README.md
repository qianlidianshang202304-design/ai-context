# AI Context 文件系统

> 🧠 我的个人 AI 上下文仓库。AI 读取后应按照 AGENTS.md 规定的流程工作。

---

## 📂 文件结构

```
ai-context/
├── AGENTS.md          ⭐ AI 工作总控（三角色 + 分级 + 流转）
├── 工作规范.md         ⚠️ 每次必读（沟通规则、输出标准、禁止事项）
├── 任务分级.md         🔀 A/B/C 三级判断标准
├── 品味标准.md         🎨 什么是"好"的代码和方案
├── 最佳案例.md         🏆 驾驶舱参考库（已验证的最佳实践）
├── 关于我.md           👤 个人信息、工作领域
├── 我的项目.md         📁 当前项目和技术栈
├── 我的偏好.md         ⚙️ 编码偏好和工具选择
├── 工作流程.md         🔄 日常流程和Git规范
├── 项目经验.md         📚 历史项目经验和踩坑
├── 对话日志.md         📔 日常对话自动记录
└── README.md           📖 本说明
```

---

## 🔄 工作流总览

```
开始 → 必读工作规范 → CEO拆解分级 → A直接干/B读md规划/C确认后方案

```
