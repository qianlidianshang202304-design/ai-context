## 项目结构偏好

> 我喜欢的目录结构和命名规范。

```
项目名/
├── backend/           # FastAPI 后端
│   ├── main.py        # 入口
│   ├── routers/       # API 路由
│   ├── models/        # 数据模型
│   └── database.py    # 数据库配置
├── frontend/          # Next.js 前端
│   ├── app/           # App Router 页面
│   ├── components/    # 组件
│   └── lib/           # 工具库
├── scripts/           # 数据处理脚本
├── .env.example       # 环境变量模板
├── docker-compose.yml # 容器编排
└── README.md
```
