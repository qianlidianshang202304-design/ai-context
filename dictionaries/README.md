## 新增字典规则

新增或修改正式字典前，必须先获得用户明确确认。

### 禁止直接写入正式字典

- AI 不允许自行把新分类、新标签、新口径写入 `dictionaries/`。
- 未确认内容只能放到 `../drafts/`，文件名必须带 `_待确认`。
- 用户明确说“确认”“可以写入”“写进字典”后，才允许从草稿进入正式字典。

### 需要先问用户确认的情况

凡涉及以下内容新增或修改，都必须先让用户审核：

1. 观众 / 人群 / 受众 / audience
2. 达人类型 / creator_type / influencer type / creator archetype
3. 内容角度 / messaging angle / Primary Messaging Angle / 视频主卖点 / 广告主诉求
4. 字段映射、标签口径、置信度规则、证据规则

### 新增字典前先确认

1. 这个分类是否会跨项目复用。
2. 是否需要统一字段名、标签名、置信度、证据口径。
3. 是否会影响报告、Excel、脚本或后续 AI 判断。
4. 用户是否已经确认最终口径。

新增字典建议命名：

```md
03-内容角度字典.md
04-字段映射字典.md
05-品牌话术字典.md
```

每个字典至少包含：

```md
定义
允许值
禁止值
证据规则
输出字段
示例
新增/修改规则
最后更新日期
```
