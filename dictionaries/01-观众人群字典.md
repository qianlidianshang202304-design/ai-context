# 观众人群字典

> 本文件是所有 audience / 受众 / 人群 / viewer / consumer segment 分类的唯一字典。
> 后续凡涉及达人分析、视频分析、品牌合作画像、竞品内容分析、广告话术分析，都必须优先读取并遵守本文。
> 未经用户明确确认，AI 不允许自行新增或替换一级、二级观众分类。

---

## 一、总原则

### 1. 一级观众人群只能从 5 类中选择

`audience_primary` 只能使用以下 5 个标签：

1. `Southern California Wellness Lifestyle`
2. `Bay Area Tech/Startup Lifestyle`
3. `East Coast Honest Evidence Based Lifestyle`
4. `South Florida Retiree Lifestyle`
5. `Southwest Family-oriented（Familismo）Lifestyle`

### 2. 二级观众标签只能从 6 类中选择

`audience_secondary` 只能使用以下 6 个标签：

1. `30's to 40's Caucasian`
2. `40's to 50's Caucasian`
3. `50+ Caucasian`
4. `30's to 40's Hispanic`
5. `40's to 50's Hispanic`
6. `50+ Hispanic`

### 3. 拼写规范

- 正式输出统一使用 `Caucasian`。
- 如果输入中出现 `Cauasian`，自动规范化为 `Caucasian`。
- 年龄格式统一使用：
  - `30's to 40's`
  - `40's to 50's`
  - `50+`

### 4. 观众分类是策略画像，不是事实人口统计

`audience_primary` 和 `audience_secondary` 默认都是品牌投放策略分类 / lifestyle archetype，不等于真实后台受众地理、年龄或族裔分布。

只有在有后台数据、品牌方受众报告、平台人群数据、调研数据或达人明确自述时，才能写：

```md
demographic_verified: Yes
```

否则必须写：

```md
demographic_verified: No
```

### 5. 内容主题不等于一级观众人群

以下内容主题不能作为 `audience_primary` 或 `audience_secondary` 的新标签：

- GLP / Weight Loss
- Mom Lifestyle
- Fitness
- Deal Buyer
- TikTok Shop Buyer
- Evidence-led Wellness
- Spanish-language Audience
- Beauty / Body Confidence
- Professional / Busy Workday

这些内容只能放在辅助字段：

```md
audience_basis
messaging_angle
content_motivation
creative_signal
uncertainty
```

## 二、标准输出字段

凡涉及观众分类，必须输出以下字段：

```md
audience_primary:
audience_secondary:
audience_basis:
evidence_source:
confidence:
demographic_verified:
uncertainty:
```

字段含义：

| 字段 | 含义 |
|---|---|
| `audience_primary` | 只能填 5 个一级观众分类之一 |
| `audience_secondary` | 只能填 6 个二级观众标签之一 |
| `audience_basis` | 为什么归到这个人群 |
| `evidence_source` | Bio / 标题 / 字幕 / 画面 / 评论 / 后台数据 / 品牌文件 |
| `confidence` | `High` / `Medium` / `Low` |
| `demographic_verified` | `Yes` / `No` |
| `uncertainty` | 仍然不确定的部分 |

Excel 字段建议：

```md
观众类型
观众二级标签
观众判断依据
观众证据来源
观众置信度
是否验证真实人口统计
仍然不确定性
```

## 三、证据优先级

### Level 1：强证据

可以支持 `Medium` / `High` 置信度：

- 品牌方明确指定目标人群
- 平台后台受众数据
- 达人 Bio 明确地区、生活阶段、职业、家庭状态
- 视频口播明确自述身份或生活方式
- 画面内文字明确点名目标人群
- 多条视频长期稳定服务同一类人群
- 评论区大量同类人群自述

### Level 2：中证据

通常支持 `Medium` 置信度：

- 多条视频标题 / hashtag 指向同一内容动机
- 达人主页长期内容主题一致
- 视频场景、口吻、产品使用方式稳定
- 内容语言和文化符号稳定
- 商品品类与话术长期一致

### Level 3：弱证据

只能支持 `Low` 置信度：

- 单条标题
- 单个 hashtag
- 单张画面
- 昵称
- AI 主观感觉
- 没有来源的推断

## 四、五类 audience_primary 定义

## 1. Southern California Wellness Lifestyle

### 中文定义

南加州式轻健康生活方式人群。

### 核心画像

这类人群重视日常健康、轻盈感、外形状态、wellness routine、自我管理、生活美学和社交表达。内容更容易被「好看、好喝、好坚持、看起来健康、适合日常 routine」打动。

### 典型内容气质

- 明亮自然光
- 厨房 / 车里 / 健身后 / 早晨 routine
- 轻松、漂亮、干净、生活方式化
- 产品像日常习惯，而不是严肃治疗方案
- 强调 feeling good / looking good / easy routine

### 常见内容信号

- wellness
- daily routine
- healthy habits
- glow
- clean girl
- pilates / yoga / walk
- gut health
- beauty from within
- self-care
- morning routine
- reset
- feel lighter
- easy healthy swap

### 适合产品角度

- 日常 wellness routine
- 轻负担补充
- 口感好、方便坚持
- 外形状态 / 轻盈感
- 美容健康结合
- 生活方式推荐
- “我每天都会喝 / 吃这个”

### 不适合写成

- 真实观众在 Southern California
- 观众一定是年轻白人女性
- 观众一定有高收入
- 产品能让人变美 / 变瘦 / 治疗问题

### 判断规则

| 置信度 | 条件 |
|---|---|
| High | Bio / 长期内容明确是 wellness lifestyle、pilates、self-care、beauty wellness，并且视频画面也是日常 routine |
| Medium | 多条视频呈现轻健康、日常补充、生活方式推荐 |
| Low | 只有单条标题或画面看起来像 wellness |

### 示例输出

```md
audience_primary: Southern California Wellness Lifestyle
audience_secondary: 30's to 40's Caucasian
audience_basis: 达人内容围绕 wellness routine、轻健康补充和生活方式推荐；视频画面为明亮厨房冲泡场景
evidence_source: TikTok Bio + 视频标题 + 关键帧
confidence: Medium
demographic_verified: No
uncertainty: 没有后台受众数据，无法验证真实地域、年龄和族裔结构
```

## 2. Bay Area Tech/Startup Lifestyle

### 中文定义

湾区科技 / 创业效率生活方式人群。

### 核心画像

这类人群重视效率、理性选择、时间管理、工作状态、精力维持、低摩擦健康习惯。重点是内容逻辑更像 busy professional / founder / productivity / evidence-aware consumer。

### 典型内容气质

- 讲逻辑、讲效率、讲机制
- 产品是 busy day 里的省事方案
- 强调 on-the-go、workday、travel、commute
- 健康习惯要简单，不增加决策成本
- 对成分、功能、对比有兴趣，但不一定需要深度医学背书

### 常见内容信号

- busy day
- workday
- office
- commute
- on the go
- entrepreneur
- founder
- productivity
- routine optimization
- travel day
- no time
- quick protein
- easy supplement
- efficient wellness
- smart choice

### 适合产品角度

- 省时间
- 方便携带
- 工作日可坚持
- 简单冲泡
- 功能清楚
- 成分 / 功能点明确
- 不占用精力的健康选择

### 不适合写成

- 真实观众在 Bay Area
- 观众一定是科技从业者
- 观众一定是高收入人群
- 只因视频里有电脑就归为这个人群

### 判断规则

| 置信度 | 条件 |
|---|---|
| High | Bio / 口播明确职业、创业、office、busy professional，且内容长期围绕效率型健康习惯 |
| Medium | 多条视频强调 on-the-go、workday、busy routine、效率补充 |
| Low | 单条标题出现 busy / work，但缺少主页证据 |

### 示例输出

```md
audience_primary: Bay Area Tech/Startup Lifestyle
audience_secondary: 30's to 40's Caucasian
audience_basis: 视频强调 hectic schedule、on-the-go 补充和低摩擦健康习惯，适合效率型生活方式人群
evidence_source: 视频标题 + 画面场景 + CTA
confidence: Medium
demographic_verified: No
uncertainty: 没有职业、地域和后台受众数据，不能确认真实科技/创业人群占比
```

## 3. East Coast Honest Evidence Based Lifestyle

### 中文定义

东海岸诚实证据型健康生活方式人群。

### 核心画像

这类人群重视真实体验、证据、解释、对比、直接表达和可信度。她们不容易被纯氛围种草打动，更在意 “为什么有用 / 成分是什么 / 我踩过什么坑 / 我真实学到了什么”。

### 典型内容气质

- 语气直接、诚实、有判断
- 不过度包装
- 愿意讲副作用、缺点、对比、失败经验
- 常见 what I learned / what nobody tells you / honest review
- 更接受专业人士、长期使用者、真实 journey 叙事

### 常见内容信号

- honest review
- what I learned
- nobody tells you
- side effects
- compare
- doctor / dietitian / nutritionist
- evidence
- ingredients
- why this works
- myth vs fact
- not medical advice
- real results
- real talk
- before / after
- pros and cons

### 适合产品角度

- 成分解释
- 机制解释
- 对比评测
- 真实经验
- 长期使用感受
- 解决一个具体痛点
- “不是神药，但这件事帮我坚持下来”

### 不适合写成

- 真实观众在 East Coast
- 有科普语气就说达人是医生
- 把 AI 总结包装成科学事实
- 没有证据却写 evidence-based

### 判断规则

| 置信度 | 条件 |
|---|---|
| High | 专业身份 / 长期真实 journey / 多条内容稳定做解释、对比、证据型表达 |
| Medium | 标题和画面明显是 honest review、what I learned、side effects、comparison |
| Low | 单条标题出现 honest / science，但无长期证据 |

### 示例输出

```md
audience_primary: East Coast Honest Evidence Based Lifestyle
audience_secondary: 40's to 50's Caucasian
audience_basis: 视频开头直接讲 side effects，画面展示 Protein / Electrolytes / Collagen 三点解释产品角色，属于直接证据型内容
evidence_source: Top5 关键帧 + 视频标题 + TikTok Bio
confidence: High
demographic_verified: No
uncertainty: 不能验证真实地域、年龄或族裔，只能判断内容风格适合证据型健康人群
```

## 4. South Florida Retiree Lifestyle

### 中文定义

南佛州退休 / 熟龄健康生活方式人群。

### 核心画像

这类人群关注熟龄健康、体重管理、活力、日常补充、医生建议、生活质量、轻松社交和长期可坚持的健康习惯。核心不是 “退休” 这个身份本身，而是内容更适合 50+ / midlife / healthy aging / GLP journey / active older lifestyle。

### 典型内容气质

- 语气更生活化、经验型
- 强调长期坚持，而不是快速变美
- 常见 50+、midlife、GLP、menopause、healthy aging
- 画面可能是厨房、家里、日常服装、前后对比
- 更信任真实体验、时间长度和生活质量改善

### 常见内容信号

- 50+
- in my 50s
- over 50
- over 40
- midlife
- menopause
- healthy aging
- GLP journey
- weight management
- active aging
- energy
- joint health
- long-term routine
- what I learned after X years
- retired / empty nester

### 适合产品角度

- 熟龄健康 routine
- 蛋白补充
- GLP 期间营养支持
- 长期坚持
- 活力和生活质量
- 简单易喝 / 易消化 / 不麻烦
- 医疗边界清楚的健康支持

### 不适合写成

- 真实观众在 South Florida
- 观众一定是 retiree
- 观众一定是 Caucasian
- 产品治疗 GLP 副作用、掉发、肌肉流失
- 只看脸判断 50+

### 判断规则

| 置信度 | 条件 |
|---|---|
| High | Bio 或口播明确 50+ / in my 50s / menopause / retired / GLP journey |
| Medium | 多条内容稳定围绕 midlife、healthy aging、GLP、长期健康管理 |
| Low | 只从画面感觉判断年龄，不能进结论摘要 |

### 示例输出

```md
audience_primary: South Florida Retiree Lifestyle
audience_secondary: 50+ Caucasian
audience_basis: Bio 明确写 in my 50's with the help of a GLP；视频讲 2.5 年 GLP 经验，并展示日常冲泡产品
evidence_source: TikTok Bio + 视频标题 + Top5 关键帧
confidence: High
demographic_verified: No
uncertainty: 只能确认达人自述 50's 和 GLP journey，不能确认真实观众地域、退休状态或族裔
```

## 5. Southwest Family-oriented（Familismo）Lifestyle

### 中文定义

美国西南家庭导向 / Familismo 生活方式人群。

### 核心画像

这类人群重视家庭、亲密关系、照顾家人、家庭采购、妈妈/女性在家庭健康中的决策角色，以及西语/双语/Latina 文化中的 familismo 价值观。重点是“家庭导向”和“照顾家人的健康选择”，不是简单等同于 Hispanic 人群。

### 典型内容气质

- 家庭、妈妈、孩子、伴侣、日常照顾
- 语气亲近、真实、像朋友分享
- 可能有西语 / 双语内容
- 强调 “我家里会用 / 我给自己和家人都考虑”
- 健康选择不是个人炫耀，而是家庭 routine 的一部分

### 常见内容信号

- mom
- mama
- motherhood
- family
- kids
- husband / wife
- busy mom
- family routine
- household finds
- Spanish
- bilingual
- Latina
- para mi familia
- todos lo podemos tomar
- tratamiento
- bienestar
- familia
- #parati

### 适合产品角度

- 家庭健康 routine
- 忙碌妈妈的便捷补充
- 西语 / 双语内容
- “大家都可以喝，但某类人特别适合”
- 照顾自己才能照顾家人
- 家庭采购好物
- 口味、方便、日常性

### 不适合写成

- 真实观众一定是 Hispanic
- 真实观众一定在 Southwest
- 西语内容就等于 Hispanic women audience
- 妈妈内容就一定是 Familismo
- 只因达人是女性就归到家庭导向

### 判断规则

| 置信度 | 条件 |
|---|---|
| High | Bio 明确 mom / family / Latina / bilingual / Southwest 地区，且内容长期围绕家庭健康或西语家庭语境 |
| Medium | 视频为西语 / 双语，或多条内容围绕妈妈、家庭、照顾家人 |
| Low | 单条标题出现 mom / familia / parati，但无主页证据 |

### 示例输出

```md
audience_primary: Southwest Family-oriented（Familismo）Lifestyle
audience_secondary: 30's to 40's Hispanic
audience_basis: 视频为西语口播，画面内文字提到 tratamiento、proteína、náuseas、colágeno，标题写 TODOS lo podemos tomar，语气是家庭/熟人式健康建议
evidence_source: 视频标题 + Top5 关键帧可见文字 + TikTok Bio
confidence: Medium
demographic_verified: No
uncertainty: 可确认西语和家庭导向内容信号，但不能确认真实观众族裔、地域或家庭结构
```

## 五、六类 audience_secondary 定义

## 1. 30's to 40's Caucasian

适用于内容更像影响 30-40 岁阶段、Caucasian 策略画像的人群。常见于轻健康、wellness routine、busy professional、mom routine、body confidence 等方向。

使用时必须说明证据来源。没有后台数据时，必须写：

```md
demographic_verified: No
```

---

## 2. 40's to 50's Caucasian

适用于内容更像影响 40-50 岁阶段、Caucasian 策略画像的人群。常见于 midlife wellness、weight management、honest review、evidence-led health、family health management 等方向。

使用时必须说明证据来源。不能只凭画面年龄感或肤色判断。

---

## 3. 50+ Caucasian

适用于内容更像影响 50+、Caucasian 策略画像的人群。常见于 healthy aging、GLP journey、退休或半退休生活方式、长期健康管理、熟龄体重管理等方向。

如果达人 Bio 或口播明确 `in my 50s` / `50+`，可以提高年龄阶段置信度；族裔仍需谨慎标注是否验证。

---

## 4. 30's to 40's Hispanic

适用于内容更像影响 30-40 岁阶段、Hispanic 策略画像的人群。常见于西语/双语内容、mom routine、family wellness、workday wellness、Latina lifestyle 等方向。

西语内容只能支持 Hispanic-language creative signal；没有后台数据时，不能写成真实族裔事实。

---

## 5. 40's to 50's Hispanic

适用于内容更像影响 40-50 岁阶段、Hispanic 策略画像的人群。常见于 family health management、midlife wellness、西语健康解释、照顾自己与家庭的健康 routine 等方向。

使用时必须写清楚依据来自西语内容、Bio、品牌人群文件还是后台数据。

---

## 6. 50+ Hispanic

适用于内容更像影响 50+、Hispanic 策略画像的人群。常见于西语熟龄健康、family-oriented wellness、retiree lifestyle、GLP / weight management、长期健康习惯等方向。

如果只有西语信号但没有年龄阶段证据，不能自动选 `50+ Hispanic`。

## 六、分类决策流程

每次判断观众人群时，按这个流程：

```md
1. 先看是否有品牌方指定人群
2. 再看达人 Bio
3. 再看视频标题 / 口播 / 画面内文字
4. 再看达人长期内容主题
5. 再看评论区或后台数据
6. 最后才看单条视频画面
```

然后从 5 个 `audience_primary` 和 6 个 `audience_secondary` 中各选一个最接近的标签。

### audience_primary 决策简表

| 如果内容主要是 | 优先选择 |
|---|---|
| wellness routine、轻健康、变美、自我管理、生活方式 | `Southern California Wellness Lifestyle` |
| busy day、效率、职场、on-the-go、理性选择 | `Bay Area Tech/Startup Lifestyle` |
| honest review、what I learned、side effects、成分解释、对比 | `East Coast Honest Evidence Based Lifestyle` |
| 50+、midlife、GLP journey、healthy aging、长期健康管理 | `South Florida Retiree Lifestyle` |
| mom、family、西语/双语、家庭健康、familismo | `Southwest Family-oriented（Familismo）Lifestyle` |

### audience_secondary 决策简表

| 如果内容主要是 | 优先选择 |
|---|---|
| 轻健康、年轻/中青年生活方式、busy routine，无 Hispanic / 50+ 强信号 | `30's to 40's Caucasian` |
| midlife、40+、evidence-led、熟龄前健康管理，无 Hispanic 强信号 | `40's to 50's Caucasian` |
| 50+、retiree、healthy aging、GLP 熟龄旅程，无 Hispanic 强信号 | `50+ Caucasian` |
| 西语/双语、年轻家庭、mom routine、Latina lifestyle | `30's to 40's Hispanic` |
| 西语/双语、midlife family wellness、家庭健康管理 | `40's to 50's Hispanic` |
| 西语/双语、50+、熟龄家庭健康、retiree wellness | `50+ Hispanic` |

## 七、置信度标准

### High

满足以下至少一条：

- Bio 明确支持该 lifestyle archetype
- 多条视频长期稳定指向该人群
- 视频口播 / 画面内文字明确点名人群
- 品牌方目标人群文件明确指定
- 后台数据支持

注意：

```md
High confidence 只代表“策略分类有较强依据”，不代表真实人口统计已验证。
```

### Medium

满足以下情况：

- 标题、画面、Bio 有 2 个以上信号一致
- Top 视频证据强，但主页长期证据不足
- 有语言 / 家庭 / 50+ / GLP / workday 等明显内容线索，但缺少后台数据

### Low

以下情况只能 Low：

- 只有单条标题
- 只有单个 hashtag
- 只有昵称
- 只有画面感觉
- 只有 AI 推断
- 缺少 Bio / 主页 / 长期内容证据

`Low` 不能进入结论摘要，只能进明细或待复核。

## 八、报告写法规则

### 可以写

```md
该视频更适合 Southern California Wellness Lifestyle：画面是明亮厨房冲泡，话术强调 daily wellness routine 和轻负担补充。
```

```md
该达人更适合 South Florida Retiree Lifestyle：Bio 自述 in my 50's with the help of a GLP，视频讲长期 GLP 经验。
```

```md
该内容归入 Southwest Family-oriented（Familismo）Lifestyle：西语表达、家庭式健康建议、语气强调 todos lo podemos tomar。
```

```md
该判断是策略画像，不是已验证人口统计。
```

### 不可以写

```md
这个视频真实观众在 South Florida。
```

```md
这个达人观众是 40-50 岁 Caucasian。
```

```md
西语视频说明观众就是 Hispanic women。
```

```md
视频里有电脑，所以是 Bay Area tech audience。
```

## 九、输出模板

每条视频或达人输出时，推荐格式：

```md
audience_primary: East Coast Honest Evidence Based Lifestyle
audience_secondary: 40's to 50's Caucasian
audience_basis: 视频用 “side effects are real” 做 hook，并用 Protein / Electrolytes / Collagen 三点解释产品角色
evidence_source: 视频标题 + Top5 关键帧可见文字 + TikTok Bio
confidence: High
demographic_verified: No
uncertainty: 无后台受众数据，不能确认真实地域、年龄、族裔
```

---

## 十、新增或修改分类的规则

`audience_primary` 默认固定为 5 类，`audience_secondary` 默认固定为 6 类。

只有用户明确要求时，才能新增或修改一级、二级分类。

新增时必须写：

```md
新增标签:
标签层级: audience_primary / audience_secondary
为什么现有分类不够:
定义:
适用范围:
不适用范围:
证据规则:
示例:
是否替代旧标签:
```

未经确认，不允许 AI 自行新增 audience 分类。

---

## 十一、总规则

1. 先归入已有字典，再考虑新增。
2. `audience_primary` 只能从 5 类中选。
3. `audience_secondary` 只能从 6 类中选。
4. `Cauasian` 自动规范化为 `Caucasian`。
5. 默认 `demographic_verified: No`。
6. 没有证据时必须降低置信度，而不是发明新分类。
7. 单条标题不能支撑 `High confidence`。
8. 报告结论区只放 `Medium` / `High` 且证据清楚的人群判断。
9. 子线程审核时必须专门检查：是否把观众策略画像写成真实人口统计事实。
10. 任何输出都必须能追溯到 Bio、标题、字幕/画面内文字、截图、评论、后台数据或品牌文件。
