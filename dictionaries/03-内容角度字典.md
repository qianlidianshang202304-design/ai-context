# 内容角度字典

> 状态：已确认，正式字典。
> 本文件已由用户确认写入正式字典库。

> 本文件用于统一所有品牌合作分析、竞品内容分析、达人视频分析、广告素材分析中的 `Primary Messaging Angle` 分类口径。
> 后续凡涉及 messaging angle、内容角度、话术角度、视频主卖点、广告主诉求，都必须优先读取并使用本字典。
> 未经用户明确确认，AI 不允许自行新增、修改或删除正式内容角度。

---

## 一、总原则

### 1. primary_messaging_angle 只能从正式列表中选择

`primary_messaging_angle` 只能使用以下正式标签：

1. `Menopause Mid-section belly fat`
2. `Curb Cravings/Late night snacking`
3. `B420 Natural GLP1`
4. `GLP1 Companion/Side Effects`
5. `Gut Probiotic vs. Metabolic Probiotic`
6. `Direct competitor comparison`
7. `Bloating`
8. `Fiber Side Effects`
9. `Value for money`
10. `Vs. Physician's Choice Gummies`
11. `Regularity`
12. `User Experience`

### 2. 别名和规范化

输入或源文件中出现以下写法时，统一规范化：

| 输入写法 | 正式标签 |
|---|---|
| `GLP1 Companion` | `GLP1 Companion/Side Effects` |
| `GLP-1 Companion` | `GLP1 Companion/Side Effects` |
| `GLP1 Companion Side Effects` | `GLP1 Companion/Side Effects` |
| `B420 Natural GLP-1` | `B420 Natural GLP1` |
| `B.420 Natural GLP1` | `B420 Natural GLP1` |
| `Physician’s Choice Gummies` | `Vs. Physician's Choice Gummies` |
| `Vs Physician's Choice Gummies` | `Vs. Physician's Choice Gummies` |
| `Late night snacking` | `Curb Cravings/Late night snacking` |
| `Cravings` | `Curb Cravings/Late night snacking` |

### 3. 内容角度不是达人类型，也不是观众类型

`primary_messaging_angle` 判断的是一条视频、广告、脚本或内容素材的**主要话术角度**。

它不等于：

- 达人类型：看 `dictionaries/02-达人类型字典.md`
- 观众人群：看 `dictionaries/01-观众人群字典.md`
- 产品类别
- 真实功效证明
- 科学事实

例如：

- 一个 Weightloss Journey 达人可以讲 `Value for money`。
- 一个 Science Authority 达人可以讲 `Bloating`。
- 一个 Young Mom 达人可以讲 `Curb Cravings/Late night snacking`。

### 4. 一个内容只能有一个主角度

每条视频 / 广告 / 脚本必须优先选一个 `primary_messaging_angle`。

如果确实有多个角度，可以另设：

```md
secondary_messaging_angle:
supporting_angles:
```

但主分析、透视表、汇总图必须使用 `primary_messaging_angle`。

### 5. IG / TikTok 视频分析必须优先看内容本体

涉及已合作达人、竞品视频、广告视频复盘时，不能只根据视频标题、caption、发布文案或达人类型判断 `primary_messaging_angle`。

标准流程：

1. 下载视频或读取可访问的视频素材。
2. 用 ffmpeg 抽取音频和关键帧。
3. 用转写工具提取口播 / 字幕逐字稿。
4. 基于逐字稿中反复出现的实质内容、画面内文字和关键帧判断主角度。
5. 标题 / caption / hashtag 只能作为辅助证据，不能单独支撑 `High confidence`。

如果视频无音频、无法下载或转写失败，必须在输出中写明 `transcript_status` / `evidence_limitation`，并降低置信度；不能伪装成已经完成文字分析。

## 二、标准输出字段

凡涉及内容角度判断，必须输出：

```md
primary_messaging_angle:
secondary_messaging_angle:
messaging_angle_basis:
evidence_source:
confidence:
transcript_status:
evidence_limitation:
unclassified_reason:
proposed_new_messaging_angle:
```

字段解释：

| 字段 | 含义 |
|---|---|
| `primary_messaging_angle` | 只能填正式标签之一；无法归类时填 `Unclassified - Needs Review` |
| `secondary_messaging_angle` | 可选，辅助角度，只能从正式标签中选 |
| `messaging_angle_basis` | 为什么这么分类 |
| `evidence_source` | 逐字稿 / 字幕 / 口播 / 画面内文字 / 关键帧 / 标题 / 评论 / 品牌文件 |
| `confidence` | `High` / `Medium` / `Low` |
| `transcript_status` | 视频任务必须填：`transcribed` / `no_audio` / `download_failed` / `not_video` / `not_processed` |
| `evidence_limitation` | 如果无法下载、无法转写、无音频或只能看标题/caption，必须说明限制 |
| `unclassified_reason` | 如果无法归类，说明原因 |
| `proposed_new_messaging_angle` | 如需新增角度，先写候选名称，等待用户确认 |

## 三、证据优先级

### Level 1：强证据

可支持 `High` / `Medium`：

- 下载视频后的口播逐字稿明确重复同一主诉求
- 画面内字幕 / 自动字幕明确重复同一主诉求
- 口播或字幕明确重复同一主诉求
- 画面内文字明确写出主卖点
- 产品包装、对比表、价格牌、评论截图直接支持该角度
- 品牌方脚本 / briefing 明确要求该角度
- 多条同系列视频都围绕同一角度

### Level 2：中证据

通常支持 `Medium`：

- 标题 / caption、画面、产品露出共同指向同一角度
- hashtag 和视频场景一致
- 达人长期内容和当前视频角度一致
- 评论区集中讨论同一痛点

### Level 3：弱证据

只能支持 `Low`：

- 单个 hashtag
- 画面感觉
- AI 主观猜测
- 产品品类本身
- 只看达人类型推断内容角度

`Low` 不能进入结论摘要，只能进明细或待复核。

## 12. User Experience

### 中文定义

用户体验 / 使用体验角度。

### 核心定义

内容主诉求围绕用户真实体验、口感、方便性、包装、冲泡、携带、坚持难度、使用感受、review，而不是明确的功能痛点或竞品对比。

`User Experience` 是兜底体验类，不是“无法判断”或“达人说自己用了”的默认分类。只要逐字稿里有更明确、更高频的功能痛点或卖点，应优先归入具体角度。

### 常见信号

- my experience
- review
- taste test
- how I use it
- easy to take
- convenient
- flavor
- texture
- packaging
- routine
- first impression
- I tried it
- honest experience

### 适用场景

- 体验型 review
- 开箱 / 冲泡 / 试吃
- 口感和方便性
- 用户故事
- 没有明确更强功能角度时的默认体验类

### 判定方法

当视频看起来像普通 review 时，先统计逐字稿里最反复出现的实质内容：

- 如果反复讲 cravings / late night snack / appetite，归 `Curb Cravings/Late night snacking`
- 如果反复讲 bloating / debloat / belly discomfort，归 `Bloating`
- 如果反复讲 regular / bathroom / digestion routine，归 `Regularity`
- 如果反复讲 B420 / natural GLP-1 / GLP-1 pathway，归 `B420 Natural GLP1`
- 如果主要讲 taste / easy / convenient / how I use it / review，且没有更强功能角度，才归 `User Experience`

如果逐字稿和画面证据不足以区分，应使用 `Unclassified - Needs Review` 或低置信度，而不是为了填满分类强行写 `User Experience`。

### 不适用

- 明确竞品对比，归 competitor comparison
- 明确 price/value，归 `Value for money`
- 明确 bloating / cravings / regularity 等功能痛点

### 示例

```md
primary_messaging_angle: User Experience
messaging_angle_basis: 视频主要是试吃、口感、冲泡和日常使用体验，没有明确功能痛点或竞品比较
evidence_source: 关键帧 + 标题
confidence: Medium
```

## 五、优先级规则

当一个内容同时满足多个角度时，按以下顺序判断：

```md
1. Vs. Physician's Choice Gummies
2. Direct competitor comparison
3. B420 Natural GLP1
4. GLP1 Companion/Side Effects
5. Gut Probiotic vs. Metabolic Probiotic
6. Menopause Mid-section belly fat
7. Curb Cravings/Late night snacking
8. Fiber Side Effects
9. Bloating
10. Regularity
11. Value for money
12. User Experience
```

说明：

- 明确点名 Physician's Choice 时，优先 `Vs. Physician's Choice Gummies`。
- 直接竞品对比优先于单一功能角度。
- B420 / GLP1 / probiotic 类型教育优先于泛 gut health。
- 明确 GLP1 用户场景优先于普通 cravings / protein / wellness。
- 明确 fiber side effects 优先于普通 bloating。
- `User Experience` 是体验类兜底角度，不应抢占更具体功能角度。
- 对已合作 IG 视频复盘，`primary_messaging_angle` 应来自逐字稿、画面内文字和关键帧的综合判断；caption 只能辅助，不能替代视频内容分析。

## 六、无法归类时

如果内容无法归入正式标签，临时输出：

```md
primary_messaging_angle: Unclassified - Needs Review
messaging_angle_basis:
evidence_source:
confidence: Low
unclassified_reason:
proposed_new_messaging_angle:
```

必须写清：

```md
why_existing_angles_do_not_fit:
example_videos:
recommendation:
```

正式写入新内容角度前，必须先获得用户确认。

---

## 七、新增内容角度审核规则

当遇到新的内容角度时，AI 必须先回答：

1. 这个角度为什么不能归入现有 12 类？
2. 它和最接近的现有角度有什么边界差异？
3. 是否已经在多个视频 / 多个竞品 / 多个项目里重复出现？
4. 如果新增，正式标签名称应该是什么？
5. 会不会影响历史数据重分类？

只有用户明确回复确认后，才能新增到本字典。

## 八、报告写法规则

### 可以写

```md
该视频归入 GLP1 Companion/Side Effects，因为标题明确写 GLP side effects，画面展示产品作为 GLP 期间日常补充。
```

```md
该视频暂标为 Unclassified - Needs Review，因为主诉求是 hormone cortisol stress，目前 12 个正式角度无法覆盖，需要用户确认是否新增。
```

### 不可以写

```md
这个视频看起来像 GLP 角度。
```

```md
这个达人是减重达人，所以视频角度一定是 GLP1 Companion。
```

```md
只要讲 probiotics 就归 Gut Probiotic vs. Metabolic Probiotic。
```

```md
只要提到便秘就写治疗便秘。
```

## 九、总规则

1. `primary_messaging_angle` 默认只能从 12 个正式标签中选。
2. `GLP1 Companion` 自动规范化为 `GLP1 Companion/Side Effects`。
3. `B420 Natural GLP-1` 自动规范化为 `B420 Natural GLP1`。
4. 明确 Physician's Choice 时，优先 `Vs. Physician's Choice Gummies`。
5. 无法归类时用 `Unclassified - Needs Review`，不能自行新增正式角度。
6. 新增正式内容角度必须先获得用户确认。
7. 内容角度判断优先看下载视频后的逐字稿、字幕、画面内文字、口播和关键帧，不靠达人类型硬判。
8. 单个 hashtag 不能支撑 `High confidence`。
9. 所有分类必须写依据、证据来源、置信度。
10. 子线程审核时必须专门检查：是否把达人类型或观众人群误写成内容角度。
11. 对 IG / TikTok 视频复盘，必须记录是否完成下载、截图、转写；未完成转写时必须说明限制并降低置信度。
